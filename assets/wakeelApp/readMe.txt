تطبيق إدارة العمائر والمستأجرين
طورت تطبيقاً بسيطاً وعملياً يساعد ملاك العمائر على إدارة المستأجرين ومتابعة الإيجارات بسهولة دون الحاجة للتعامل الورقي أو الحسابات اليدوية.

مميزات التطبيق:

إدارة المستأجرين: تسجيل بيانات المستأجرين ومتابعة حالة الدفع لكل واحد منهم.

حساب الإيجارات: توضيح المستأجرين الذين سددوا والذين لم يسددوا بعد.

حساب التكاليف الإضافية: مثل استهلاك الماء والكهرباء لكل شقة أو مستأجر بشكل منفصل.

التقارير التفصيلية: حول الإيرادات، المصروفات، والمتأخرات.

إشعارات وتنبيهات آلية: إرسال رسائل نصية أو عبر واتساب للمستأجرين عند حلول موعد السداد، أو لتوضيح المبالغ المستحقة عليهم (إيجار، ماء، كهرباء).

الهدف من التطبيق هو تسهيل إدارة العقار بشكل كامل لصاحب العمارة، وتوفير الوقت والجهد مع متابعة دقيقة لحالة المستأجرين والتكاليف.
